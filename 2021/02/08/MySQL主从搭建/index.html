MySQL主从同步搭建

环境

  • docker
  • docker-compose
  • mysql:5.7.28

主库搭建

  1. 创建并进入MySQL文件夹

  2. 编写docker-compose.yaml

  3. 创建挂载的数据卷

  4. 编写MySQL配置文件my.cnf

  5. 启动并进入mysql-master

  6. 查看主节点状态信息

    注意记录此时主节点状态信息中的File字段和Position字段对应数据

从库搭建

  1. 创建并进入MySQL文件夹

  2. 编写docker-compose.yaml

  3. 创建挂载的数据卷

    注意：此时复制主节点的data目录到当前目录中

  4. 编写MySQL配置文件my.cnf

  5. 启动并进入mysql-slave

  6. 配置从节点对应主节点信息

    MASTER_HOST：主库IP MASTER_PORT：主库端口 MASTER_USER：主库用户名 MASTER_PASSWORD：主库密码 MASTER_LOG_FILE：主库节点信息中File字段对应数据 MASTER_LOG_POS：主库节点信息中Position字段对应数据

  7. 启动复制线程

  8. 查看从节点状态信息
